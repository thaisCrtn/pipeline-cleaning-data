# Data cleaning and automation pipeline

This is a constantly evolving project to automate the data wrangling process (cleaning and transformation) using Python.

## How does working 
The project is structured in a modular way:
- `src/pipeline.py`: Contains the fuctions with business and data-cleaning rules.
- `main.py `: Main Script that run the pipeline.
